# Documentação GIT

## Comandos GIT
Comando para adicionar arquivos na caixa

    git add .

Comando para fechar a caixa

    git commit -m "descrição"

Comando para status dos procedimentos

    git status

Comando para "Mandar o save para a nuvem"

    git push

Comandos em ordem para uma "caixa"

    git add. 
    git commit -m "descrição"
    git push
